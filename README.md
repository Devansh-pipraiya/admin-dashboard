# 👨🏻‍💻 Admin Dashboard

### A static dashboard layout built entirely with **CSS grid** — just nested **grids** with minimal flexbox.

This project is part of **The Odin Project** course, built to practice:

- **CSS Grid** (nested grids, fr units, grid-template-areas)
- Structuring complex **layouts** with pure **HTML** and **CSS**
- Working with **SVGs**, **positioning** and CSS _functions_ like clamp(), minmax() etc.

---

> 📌 **Project status:** Completed ✅ | 🌐 [Live Preview](https://devansh-pipraiya.github.io/admin-dashboard/)

---

## ✨ Features

- 🔲 **Nested Grid Layout** — the entire dashboard is structured with CSS Grid, grids nested inside grids for every section

- 📐 **Auto-fit Project Cards** — cards automatically resize and wrap to fit the available width using `auto-fit` and `minmax()`

- 🖼️ **Live Screenshot Preview** — hover over any project card icon to see a full pop-up screenshot of that project

- ✨ **Smooth Animations** — hover effects and scaling on buttons, nav links, and profile photos throughout the UI
